# Vajreshwari Enterprises Website

A premium, responsive single‑page website showcasing the services, portfolio and legacy of **Vajreshwari Enterprises** – a leading transportation and logistics provider in India.

## ✨ Features
- Modern, dark‑mode‑friendly UI with smooth animations
- Fully responsive layout (desktop, tablet, mobile)
- Interactive hero section with a continuous Lissajous‑drift background
- Service cards with hover effects and SVG‑style icons
- Detailed statistics and client showcase sections
- Accessible, SEO‑optimised markup (semantic HTML, proper meta tags)

## 🛠️ Tech Stack
- **HTML5** – structure & semantics
- **CSS3** – custom design system, gradients, glass‑morphism, media queries
- **Vanilla JavaScript** – hero animation, scroll effects

## 📦 Repository Structure
```
📂 images/          # Hero banner & UI assets
📄 index.html       # Home page (entry point)
📄 about.html       # About page
📄 services.html    # Services overview
📄 clients.html     # Clients & coverage
📄 contact.html     # Contact form / details
📄 styles.css       # Global stylesheet
📄 animations.css   # Animation‑specific CSS
📄 main.js          # Core JavaScript logic
📄 README.md        # This file
```

## 🚀 Local Development
1. **Clone** the repo (if you haven’t already):
   ```bash
   git clone https://github.com/ilhan-freelance/vajreshwari-enterprises.git
   cd vajreshwari-enterprises
   ```
2. **Start a local server** (any static file server will do):
   ```bash
   npx -y serve . -p 3000
   ```
   Open <http://localhost:3000> in your browser to view the site.
